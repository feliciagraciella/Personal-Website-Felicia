This application is a Tic Tac Toe game application. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner. It is a multiplayer game, so two players can play against each other consecutively.

I made this application to complete This project was created to complete the Algorithm and Programming Final Project. My role in this team was a programmer.

The challenge in making this project was to make this game can be played in two devices at the same time. The logic of the game itselfwas not that hard, but it was challenging how to connect the two devices. At that time also, because of the pandemic situation, we could not meet our team, so we could not try the multiplayer aspect. In the end, we succeed to make this application can be played in two windows (one device).
